Побудувати сопрограму(и) та демо-код їх роботи (якщо це не зазначено у варіанті
явно). Здається: весь розроблений код, скріншот компіляції та збирання, скріншоти
роботи демо-програми.

16. Написати сопрограму, що читає з стандартного входу дійсні числа і повертає
середнє арифметичне трьох останніх. Нулі у вхідній послідовності ігноруються, проте
за надходження нуля сопрограма має зупинятися і не брати цей нуль до уваги.
Сопрограма не повинна дозволити себе відновити, якщо з моменту отримання нуля
пройшло менше 1 секунди. Блокувати чи засипати сопрограму не можна.
